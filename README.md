# 📊 Data Analytics Journey | 5-Month Challenge

Hi there! I'm a student at **Corvinus University of Budapest** studying Applied Economics. 
I've set a goal to self-study and transition into a **Data Analyst** role within 5 months (Jan 2026 - June 2026). 
This repository serves as my public portfolio, documentation of my technical growth, and a log of my daily progress.

## 🚀 Learning Roadmap

<details>
<summary><b>✅ Stage 1: SQL & Data Processing </b></summary>

* **Week 1-2**: Filtering, Sorting, Aggregations, and all types of `JOIN`s (SQL Academy & Stepik).
* **Week 3-4**: Advanced Analytics: **Window Functions** (`RANK`, `ROW_NUMBER`) and **CTEs** (`WITH` statements).
* **Week 5-6**: **Excel & Power Query**: ETL processes, Unpivot, `XLOOKUP`, and Interactive Dashboards.
</details>

<details>
<summary><b>✅ Stage 2: Python for Data Science </b></summary>

* **Week 7-9**: Python Fundamentals: Data types, loops (`FOR`/`WHILE`), and conditional logic.
* **Week 9-11**: Advanced Basics: String manipulation, Lists, and Functions.
* **Week 12-13**: **Pandas & NumPy**: Vectorized operations, DataFrames, loc/iloc, and handling missing values (NaN).
</details>

<details>
<summary><b>✅ Stage 3: Statistics & Product Analytics </b></summary>

* **Week 16**: **Intro to Statistics**: Linear Regression, Decision Trees, and Classification.
* **Week 17**: **Product Metrics**: Retention, LTV, Churn Rate, and Unit Economics.
</details>

<details open>
<summary><b>🟡 Stage 4: Data Visualization </b></summary>

* **Week 14**: Mastering **Tableau Public** & **Plotly** for interactive charts.
* **Week 15**: Visualization Portfolio Project: Building a dashboard based on previous SQL/Python stages.
</details>

<details>
<summary><b>⚪ Stage 5: Final Capstone & Career </b></summary>

* **Week 18-19**: **End-to-End Capstone Project**: Predicting customer churn or sales forecasting.
* **Final Step**: CV optimization, LinkedIn networking, and internship applications.
</details>
